# Ashley - Creative Agency Website

A modern, animated creative agency website template with smooth transitions, custom cursor effects, and portfolio showcases.

## Features

- Custom animated cursor with hover effects
- Smooth page transitions using Swup
- Portfolio galleries with lightbox (Fancybox)
- Testimonial sliders
- Team member profiles
- Blog/publication sections
- Contact forms
- Fully responsive design
- GSAP scroll animations
- Swiper sliders for carousels

## Pages

- **Home Pages**: home-1.html (Landing), home-2.html (Personal)
- **Portfolio**: portfolio-1.html, portfolio-2.html, portfolio-3.html
- **Projects**: project-1.html through project-6.html
- **Services**: services.html, service.html
- **Blog**: blog.html, publication.html, blog-inner.html
- **Other**: team.html, contact.html, 404.html

## Development

To run the development server:

```bash
npm install
npm run dev
```

The site will be available at http://localhost:5173

## Build

To build for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Technologies

- HTML5
- CSS3/SCSS
- JavaScript (jQuery)
- GSAP (Animations)
- Swiper (Sliders)
- Swup (Page Transitions)
- Fancybox (Lightbox)
- Bootstrap Grid
- Font Awesome Icons
- Vite (Build Tool)
